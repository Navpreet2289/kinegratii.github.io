```
>>> import operator
>>> from functools import reduce
>>> t = [b'\x01',b'\x01\x01']
>>> reduce(operator.add,t,b'')
b'\x01\x01\x01'
>>> ^C
```

连接二进制数据应该使用 `b''.join(t)` 或者 `reduce(operator.add,t,b'')`，`sum`函数应该只支持数字类型的。
